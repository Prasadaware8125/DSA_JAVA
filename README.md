## 🚀 Getting Started

### Prerequisites

To compile and run the source code files in this repository, make sure you have the Java Development Kit (JDK) installed on your system.

* **Java Version**: JDK 8 or higher (JDK 17+ recommended)
* Check if Java is installed by running:
  ```bash
  java -version
  ```

### How to Compile and Run a Program

Each subdirectory contains raw `.java` files containing a `main` method. You can compile and run them directly from your command line:

1. Open your terminal or Command Prompt.
2. Navigate to the specific directory where the file is located:
   ```bash
   cd "01_Arrays"
   ```
3. Compile the Java file:
   ```bash
   javac KadanesAlgorithm.java
   ```
4. Run the compiled class:
   ```bash
   java KadanesAlgorithm
   ```
